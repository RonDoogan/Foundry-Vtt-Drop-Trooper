# Reaction System Phase 2 — Overwatch HUD Button

This pass adds a HUD button for toggling Overwatch on actors that have reactions enabled.

## Included
- HUD `OW` button for single-actor HUD
- button disables when reactions are unavailable
- active styling + `OW On` label when armed
- chat feedback when Overwatch is armed or canceled

## Current scope
- no movement-triggered fire yet
- no brace / return fire prompts yet
- no action-spend enforcement yet
